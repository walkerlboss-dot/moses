# BOOTSTRAP.md — Moses

## Bootstrap Checklist

Run these steps in order. Each step must pass before proceeding.

### Step 1: Verify Environment

```bash
# Confirm Python
python3 --version  # Expect: 3.14.3

# Confirm architecture
uname -m  # Expect: arm64

# Confirm writable workspace
mkdir -p ~/.openclaw/workspace/moses-builds
touch ~/.openclaw/workspace/moses-builds/.bootstrap_test
rm ~/.openclaw/workspace/moses-builds/.bootstrap_test
```

**Gate:** If any check fails, stop and report to Alex.

### Step 2: Initialize Git Repo

```bash
cd ~/.openclaw/workspace/moses-builds
git init
git config user.name "Moses Builder"
git config user.email "moses@boss.industries"

# Create directory structure
mkdir -p {cad,code,sim,tests,docs,bom,pcb,logs}

# Initial commit
git add .
git commit -m "Bootstrap: Moses build environment initialized"
```

### Step 3: Install Core Packages

```bash
# Physics simulation
pip install mujoco
pip install mujoco_menagerie

# ML/RL
pip install torch torchvision
pip install stable-baselines3
pip install gymnasium

# Testing
pip install pytest pytest-cov hypothesis

# Utilities
pip install numpy scipy matplotlib pandas
pip install GitPython
pip install opencv-python

# Verification
python3 -c "import mujoco; print('MuJoCo:', mujoco.__version__)"
python3 -c "import torch; print('PyTorch:', torch.__version__)"
python3 -c "import pytest; print('pytest:', pytest.__version__)"
```

**Gate:** All imports must succeed. If any fail, retry once, then report.

### Step 4: Create .gitignore

```bash
cat > ~/.openclaw/workspace/moses-builds/.gitignore << 'EOF'
# Python
__pycache__/

# Large binaries
*.pt

*.onnx

# Secrets

*.key
EOF

git add .gitignore
git commit -m "Bootstrap: Add .gitignore"
```

### Step 5: Create README

```bash
cat > ~/.openclaw/workspace/moses-builds/README.md << 'EOF'
# moses-builds

Autonomous humanoid robotics build artifacts.
Generated by Moses, reviewed by Alex.

## Directory Structure
- `cad/` — CAD files, URDFs, mechanical designs
- `code/` — Control software, ROS2 nodes, utilities
- `sim/` — Simulation scripts, environments, results
- `tests/` — Test suites, benchmarks
- `docs/` — Design documents, analyses
- `bom/` — Bills of materials
- `pcb/` — Circuit schematics, PCB layouts
- `logs/` — Build logs, simulation outputs

## Rules
- Every commit has tests.
- Every design has simulation.
- Every artifact is versioned.
EOF

git add README.md
git commit -m "Bootstrap: Add README"
```

### Step 6: Verify Tool Chain

```bash
# Test MuJoCo
python3 -c "
import mujoco
model = mujoco.MjModel.from_xml_string('<mujoco><worldbody><body><freejoint/><geom size=\"0.1\" type=\"sphere\"/></body></worldbody></mujoco>')
data = mujoco.MjData(model)
mujoco.mj_step(model, data)
print('MuJoCo test: PASS')
"

# Test PyTorch
python3 -c "
import torch
x = torch.randn(10, 10)
y = torch.matmul(x, x.T)
print('PyTorch test: PASS')
"

# Test pytest
python3 -c "
import pytest
print('pytest test: PASS')
"
```

**Gate:** All tests must print PASS.

git log --oneline
